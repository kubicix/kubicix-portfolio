🦮SmartNavigation - Accessibility App
Project Mission

SmartNavigation is an Android application designed specifically for visually and hearing impaired users to navigate to bus stops using voice commands and real-time directions. This project demonstrates my commitment to developing inclusive technology solutions.

Accessibility Features
  • Voice command recognition for hands-free operation
  • Audio feedback for navigation instructions
  • Real-time GPS tracking and route optimization
  • Integration with public transportation data
  • Large text and high contrast UI elements
  • Vibration feedback for important notifications
  • Offline map capabilities for areas with poor connectivity
Social Impact

This application addresses a real-world need by providing enhanced mobility solutions for individuals with disabilities, promoting independence and accessibility in public transportation.
